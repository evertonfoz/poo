# Registro de Sessoes

Este diretorio centraliza o historico de sessoes de trabalho do projeto.

## Convencao de arquivos

- `sessao-001.md`
- `sessao-002.md`
- `sessao-003.md`
- `sessao-template.md`

## Template

Use `sessao-template.md` como base para criar novos registros de sessao.

Padrao sugerido por arquivo:

- Data
- Objetivo
- Decisoes tomadas
- Alteracoes realizadas
- Status final
- Proximos passos
